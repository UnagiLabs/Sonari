# Nautilus Membership Verifiers

This directory is a placeholder for future residence and student membership verifiers.

The current tree intentionally contains only shared TypeScript placeholder contracts and README
stubs. It does not define verifier logic, schemas, migrations, request formats, TEE execution, or
Sui submission behavior.

Future common Nautilus components such as `runner`, `relayer`, and reusable `shared` utilities can
move under `nautilus/` when more verifier families need them.
