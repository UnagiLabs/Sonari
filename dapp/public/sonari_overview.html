Sonari MVP Specification

Verified disaster relief on Sui

Sonari turns donation pools into programmable disaster relief. Nautilus verifies disasters and identity status. Sui Move pays verified Membership SBT owners under clear rules.

1. まず伝えたい設計

保険ではない

Sonari は支払いを保証しません。寄付 Pool、災害条件、本人確認、予算上限に基づいて支援します。

KYC または World ID

Membership SBT は、KYC または World ID のどちらかで確認済みになると Claim できます。

SBT owner に支払う

MVP の受取先は Membership SBT owner の Sui address です。別の外部受取先は持ちません。

2. Claim 対象条件

分類 条件
災害 DisasterEvent が finalized 済みである
SBT Membership SBT が active である
災害前条件 account_created_at_ms と home_cell_registered_at_ms が cutoff より前である
地域 登録済み home cell が affected cells に含まれる
本人確認 identity_verified == true である
重複防止 duplicate key がこの Membership SBT に紐づいている

3. 支給率

未認証
  -> Claim 不可

KYC verified
  -> 100% Claim 可

World ID verified
  -> 100% Claim 可

KYC と World ID は同じ満額ルートです。 本人確認の種類で支払額を減らしません。

4. Nautilus が検証するもの

  • Earthquake payload と affected cells
  • KYC provider response と署名
  • World ID の Sonari 専用 action
  • provider 内 duplicate key
  • evidence hash と expiry

生の本人情報、credential の詳細、document image、World ID proof detail は on-chain に出しません。

5. 支払いフロー

  1. スポンサーが寄付する

    寄付は Main Pool または Designated Relief Pool に入ります。

  2. Nautilus が災害を検証する

    地震 event、発生時刻、affected cells root を検証します。

  3. Nautilus が本人確認を検証する

    KYC または World ID の result を Membership SBT に反映します。

  4. ユーザーが Claim する

    Move が cutoff、home cell、本人確認、duplicate claim を検証します。

  5. SBT owner へ支払う

    支払いと ClaimReceipt の作成を同じ流れで実行します。

6. 二重利用への対応

KYC と World ID をまたいだ同一人物判定は、MVP では完全には行いません。 代わりに、登録時と Claim 時に明示的な確認を表示します。

  • 他に有効な Sonari Membership SBT を持っていないこと
  • 同じ災害で複数の Membership SBT から Claim しないこと
  • 虚偽申告や複数 Claim は停止や返還請求の対象になり得ること

この内容に対して Sui wallet 署名を求めます。

7. MVP で作るもの

  • Donation Pool と透明な残高表示
  • Earthquake DisasterEvent の検証
  • KYC / World ID による Membership SBT verification
  • Membership SBT owner への Claim
  • ClaimReceipt と sponsor impact 表示

8. MVP で作らないもの

  • 別の外部受取先
  • 銀行口座への送金
  • provider をまたぐ完全な同一人物判定
  • 複雑な treasury strategy の実入金
  • 本番法務運用
